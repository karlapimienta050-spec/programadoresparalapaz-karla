Sistema operativo:
Es el software principal del computador o servidor. Administra los recursos del hardware y permite ejecutar aplicaciones.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear aplicaciones y servidores web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y la lógica de funcionamiento de la aplicación.

Puerto:
Es un canal de comunicación que permite al servidor recibir solicitudes. En este caso se utiliza el puerto 3000.

Logs:
Son registros que almacenan información sobre la actividad y eventos del servidor. Ayudan a detectar errores y monitorear el funcionamiento del sistema.
